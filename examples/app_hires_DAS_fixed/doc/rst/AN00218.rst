Overview
--------

This demo application shows a simple DAS bemformer in high resolution mode.


|appendix|
|newpage|

Demo Hardware Setup
------------------- 

Ensure that an XTAG is connected along with the microphone array via USB. Headphones hsould be connected to the audio jack for listening to the beamformed output.

|newpage|

Launching the demo application
------------------------------

Build the application then run using ``xrun --xscope app_hires_DAS_fixed.xe``. Buttons ``A`` and ``D`` rotate the direction of the beam which is indicated by the LEDs.
