Overview
--------

This demo application shows a simple Delay and Sum (DAS) beam-former.


|appendix|
|newpage|

Demo Hardware Setup
------------------- 

Ensure that an XTAG is connected along with the microphone array via USB. Headphones should be connected to the audio jack for listening to the beamformed output.

|newpage|

Launching the demo application
------------------------------

Build the application then run using ``xrun --xscope example_lores_DAS_fixed.xe``. Buttons ``A`` and ``D`` rotate the direction of the beam which is indicated by the LEDs.
